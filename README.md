# angular_barcode

Draw Barcode or QrCode with Angular 2 dart

## Usage

A simple usage example:

```html
<angular-barcode value="1234567890128" format="ean13"></angular-barcode>
<angular-qrcode data="1234567890128" size="200" color="0000ff"></angular-qrcode>
```
